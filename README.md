### **Investigation of M-protein auto-immunogenicity**
ðŸ”— https://github.com/bkhimek/M_protein_autoimmunogenicity

An in-silico study of **Streptococcus pyogenes M-protein** variation and its potential role in
**post-infectious autoimmunity**, including rheumatic fever.

**Key elements:**
- MAFFT-based sequence alignment and region curation
- Sliding-window peptide generation (15-mers)
- Quantitative molecular mimicry scoring against human cardiac proteins
- Strain-specific (emm-type) comparison
- **MHC class II prioritisation using DeepMHCII**
- Script-only, reproducible pipeline

This project demonstrates how **sequence similarity, immunological context, and population genetics**
can be combined to generate testable hypotheses.
